# Ansible Role: elk

An Ansible Role that installs Elasticsearch on Debian/Ubuntu.

**Tested on Ubuntu 16.04**


## Getting started

Make sure that python:2.7 is installed on the host.

Look at `defaults/main.yml` and `vars/main.yml` for variables.
